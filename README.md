# Be Your Own Furniture Maker

A website showcasing affordable, attractive, portable DIY home furniture designs from the "Be Your Own Furniture Maker" book series.

## Hosting

This site is hosted via GitHub Pages. To deploy:

1. Push this repository to GitHub
2. Go to Settings → Pages
3. Set source to "Deploy from a branch" → main → / (root)
4. Your site will be live at `https://<username>.github.io/<repo-name>/`
